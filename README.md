# akasha-prospector

Akasha Prospector is the directive and hypothesis layer of the Akasha ecosystem.

It reads patterns surfaced by akasha-attractor and converts them into disciplined
next-step directives for the rest of the Akasha stack.

Stack:

akasha-grand-atlas → akasha-apis → akasha-time-nexus → akasha-events → akasha-attractor → akasha-prospector

Prospector closes the research loop by proposing what to investigate next.
